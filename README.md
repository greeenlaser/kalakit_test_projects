This is a repository that collects [KalaKit](https://github.com/KalaKit) test projects to help you learn how to use them with real working example projects following the up to date code and binaries from their libraries. Each folder is fully standalone and dedicated only to the target library.

## Game-test

This project utilizes KalaWindow, KalaUI, KalaPhysics, KalaAudio and several other custom scripts to make games with the KalaKit libraries.

---

## KalaLua

This project is a very simple showcase of how to use KalaLua translation layer beteen C++ and Lua code.

---

## UI-test

This project utilizes KalaWindow and KalaUI to test out how UI functions and how you can interact with it.
